# README

## Overview

Find a random GIF using the [Giphy][] API.

## Pre-requisites

This project depends on [Node.js][].

## Install

```shell
npm install
```

## Build

To build the app:

```shell
npm run build
```

Now open the app in a browser:

```shell
open dist/index.html
```

## Test

To run the app's tests:

```shell
npm test
```

[giphy]: https://giphy.com/
[node.js]: https://nodejs.org/
